# mednafen-server-raspberry-pi
mednafen-server-raspberry-pi

Server for raspbian issue 8 and issue 9

# Server side
Run the server : mednafen-server standard.conf
Your server is at 192.168.123.100 


# Client side
$ mednafen
press 't'
and type: 
 /connect 192.168.123.100
